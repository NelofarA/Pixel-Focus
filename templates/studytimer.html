About Timer

Requirements: you need to have at least one skill in the flashcard section. No time inputted can be lower than 10 mins.


How it works

Click start the timer and start working, just make sure not to move your mouse! The timer dectects any changes in visiblity mouse movement, window changing, etc. So focus and study hard! If you interact with screen in any way the hazard level will go up. You have 3 chances, after losing one a cooldown will begin. If you lose your last life it is game over! Complete a session and you will get rewarded! Only completed session earn XP.


Scoring & Rewards

The more time you spend the more you earn:

  • 10 - 20 mins → 1 XP per 4 mins
  • 21 - 45 mins → 1 XP per 3 mins
  • 46 mins and greater → 1 XP per 2 mins

💡 Tip: If you accidently start a session the start timer button won't work unless you hit R, so make sure to reset!
